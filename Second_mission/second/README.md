## 2번 미션 

### 주제

- 머리글을 하단에서 표현하는 방법

### 주요 기능

  1. 결합된 축 활용
  2. 계산된 필드 만들기
  3. 레이아웃 구성
  
### 가이드

- 대시보드 크기는 1000 x 800 pixels로 지정합니다.
- 성능 향상을 위해 필터는 가급적 덜 사용하는 것이 좋습니다. 필터를 사용하지 않기 위해 2018 매출과 2019 매출을 각각 계산식으로 만들어 봅니다
